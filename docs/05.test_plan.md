# 테스트 계획

## 단위 테스트

| 케이스 | 목적 | 기대 결과 |
| --- | --- | --- |
| 재료 kcal 계산 | 중량과 100g당 kcal 계산 검증 | `weight * kcal / 100` 반환 |
| 재료 합산 | 여러 재료의 kcal 합계 검증 | 합산값 반환 |
| 음식명 누락 | 입력 검증 | 실패 메시지 반환 |
| 레시피 API 실패 | 오프라인 예외 처리 | 저장된 Room DB 캐시 표시 또는 오류 반환 |
| AI Vision 실패 | 네트워크/인식 오류 처리 | 수동 입력 UI(Fallback) 활성화 검증 |

## Room 테스트

| 케이스 | 목적 | 기대 결과 |
| --- | --- | --- |
| 음식 등록 | FoodItem 저장 검증 | 목록에서 조회됨 |
| 음식 수정 | 기존 FoodItem 갱신 검증 | 수정값 조회됨 |
| 음식 삭제 | cascade 삭제 검증 | 음식과 재료가 삭제됨 |
| 음식-재료 조회 | Relation 검증 | FoodWithIngredients 반환 |

## Compose UI 테스트

| 케이스 | 목적 | 기대 결과 |
| --- | --- | --- |
| 목록 표시 | seed 데이터 표시 검증 | 김밥/라면 표시 |
| 등록 검증 | 빈 음식명 저장 방지 | 오류 Snackbar 표시 |
| 상세 이동 | 목록-상세 네비게이션 검증 | kcal 상세 표시 |
| 삭제 | 상세 삭제 동작 검증 | 목록으로 복귀 |
| 레시피 검색 화면 | API 검색 UI 검증 | Recipe Codex 화면 표시 |
